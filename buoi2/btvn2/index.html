Những Trang Web Hay Dành Cho Front-End


04 tháng 01, 2021

Frontend Lập trình web
Với sự phát triển nhanh chóng của kỹ thuật web thì ngày càng có nhiều trang web bổ ích ra đời để giúp chúng ta có thể học tập cũng như có thể nâng cao kỹ năng lập trình thông qua các bài viết hữu ích. Nào bây giờ bạn hãy cùng mình đi vào tìm hiểu nhé!

CSS Animation

Các bài báo, thủ thuật và hướng dẫn về animation trong CSS. Nâng cao kỹ năng animation CSS của bạn thông qua các bài viết được đăng lên trang web hàng tuần.

Link:cssanimation.rocks

Learn to Code HTML & CSS

Trang web hướng dẫn mọi người cách xây dựng các trang web đẹp và trực quan bằng các bài học rõ ràng và có tổ chức thông qua việc sử dụng HTML, CSS. Các bài giảng bao gồm nhiều chủ đề thiết kế và phát triển web, từ cấp độ kỹ năng mới bắt đầu đến nâng cao.
Link:learn.shayhowe.com

Grid by Example

Nơi giúp bạn có thể dễ dàng tìm hiểu về mọi thứ của grid trong CSS một cách đầy đủ nhất. Và nó cũng cung cấp nhiều ví dụ minh họa khác nhau để giúp bạn có thể dễ dàng xây dựng hệ thống lưới cho trang web ở trong thực tế.
Link: gridbyexample.com

General Assembly Dash

Nơi giúp bạn có thể học HTML, CSS và Javascript ngay trên trình duyệt và có thể thấy được kết quả của đoạn mã mình viết thông qua những bài học ở trong chương trình.
Link: dash.generalassemb.ly

30 SECONDS OF INTERVIEWS

Đây là trang web tổng hợp những câu hỏi hay gặp trong quá trình phỏng vấn. Đồng thời nó cũng đưa ra các gợi ý để bạn có thể trả lời những câu hỏi đó một cách hợp lý nhất.
Link: 30secondsofinterviews.org

CSS In Real Life

CSS In Real Life là một blog bao gồm các chủ đề CSS và các đoạn trích hữu ích trên ngôn ngữ đẹp nhất của web.
Link: css-irl.info

Codrops

Codrops là một blog phát triển và thiết kế web xuất bản các bài báo và hướng dẫn về các xu hướng web mới nhất, các kỹ thuật và các skill mới. Nhóm Codrops luôn tận tâm cung cấp nội dung hữu ích, truyền cảm hứng và sáng tạo miễn phí. Link: tympanus.net
